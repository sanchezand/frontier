## Installation
Install with NPM or yarn
```bash
npm i frontier-css
# Or use yarn
yarn add frontier-css
```
Import Frontier's CSS in your project entry file
```js
import 'frontier-css/css/frontier.css'
```

_* Frontier is currently not hosted in any CDN service_
